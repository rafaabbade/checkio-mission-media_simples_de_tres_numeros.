Minha fórmula parece correta, mas o resultado está errado. O que pode estar acontecendo?

Verifique se você está respeitando a hierarquia das operações. Use parênteses para garantir que a soma seja calculada antes da divisão: (a + b + c) / 3.

Como garantir que a média seja um número float (número decimal)?

Você pode usar a função float() para converter a soma para decimal antes de dividir: float(a + b + c) / 3.
